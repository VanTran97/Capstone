Register and Login to database

1. Download these folders containing user.js, api.js and server.js and files into a folder.
2. Use node.js to download express, body-parser and mongoose modules in terminal. Use cmd and change the directory to the folder from step 1, then use 'npm install express body-parser mongoose --save'.
3. Use the command 'npm server' to turn on the server. It should connect to localhost and mongodb.
4. Go on web browser and enter 'localhost:3000'.
5. To test the register and login functionality, refer to this playlist 'https://www.youtube.com/playlist?list=PLC3y8-rFHvwg2RBz6UplKTGIXREj9dV0G' for better understanding. It is based on this tutorial.
